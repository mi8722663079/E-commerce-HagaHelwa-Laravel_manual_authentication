A custom-built e-commerce web application developed with Laravel. This project features a dual-panel architecture (Admin and User) and utilizes a manually implemented authentication system using Laravel's core session and guard mechanisms, bypassing automated scaffolding tools like Breeze or Jetstream.

## 🚀 Features

### User Panel
- **Custom Authentication:** Registration, login, and secure session management.
- **Product Catalog:** Browse products, filter by categories, and view detailed product pages.
- **Shopping Cart:** Add, update, and remove items dynamically.
- **Checkout System:** Secure order placement and checkout flow.
- **User Dashboard:** Order history tracking and profile management.

### Admin Panel
- **Secure Access:** Dedicated admin guard and middleware to prevent unauthorized access.
- **Dashboard Overview:** High-level metrics on sales, users, and inventory.
- **Inventory Management:** Create, read, update, and delete (CRUD) products and categories.
- **Order Management:** View, process, and update the status of customer orders.
- **User Control:** View registered users and manage their access.

## 💻 Tech Stack
- **Framework:** Laravel 10.x / 11.x
- **Language:** PHP 8.x
- **Database:** MySQL / MariaDB
- **Frontend:** Blade Templates, HTML5, CSS3, JavaScript / TailwindCSS

## ⚙️ Prerequisites
Before you begin, ensure you have the following installed on your local machine:
- PHP >= 8.1
- Composer
- MySQL or MariaDB
- Node.js & NPM (for frontend asset compilation)
